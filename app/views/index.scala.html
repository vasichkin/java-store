@() Web Development with Java course final application

Store application

User operations:

    
POST    /login                      { email, password }         Login to store
GET     /logout                                                 Logout out of the store
POST    /register                   { email, password }         Register new user
GET     /user/:user                                             Get user info
PUT     /user/:user                                             Reset password

Cart operations:


PUT     /cart/add                   { productId, quantity }     Add products to cart
PUT     /cart/remove                { productId, quantity }     Remove products from cart
GET     /cart                                                   List cart content
GET     /checkout                                               Checkout

Products operations:

    
GET     /products                                               List products
POST    /product                    { id, title, stock, price}  Add product
GET     /products:id                                            Show product info
PUT     /product/:id                { title, stock, price}      Change product details
DELETE  /product/:id                                            Delete product
